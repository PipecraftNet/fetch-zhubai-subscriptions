# fetch-zhubai-subscriptions

> 获取竹白订阅者信息的脚本

## 使用方法

1.

```sh
git clone https://github.com/PipecraftNet/fetch-zhubai-subscriptions.git
```

2.

```sh
npm install
```

3. 复制 `.env_sample` 文件到 `.env` 文件，在里面添加 `COOKIE_VALUE` 值。

4.

```sh
npm start
```
